## âš™ï¸ Installation

Clone the repo:

```bash
git clone https://github.com/yourusername/prompt-enhancer-engine.git
cd prompt-enhancer-engine
```

Install dependencies:

```bash
npm install
```

Add your Gemini API key:

```env
# .env.local
VITE_GEMINI_API_KEY=your_api_key_here
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
